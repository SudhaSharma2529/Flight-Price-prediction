# Flight-Price-prediction
Predicts domestic Indian flight ticket prices using 7 ML models trained on 10,000+ booking records.
# Flight Price Prediction Using Machine Learning

## Results
| Model | R2 Score | MAE |
|-------|----------|-----|
| XGBoost (Final) | 0.8918 | Rs.796 |
| Random Forest | 0.8774 | Rs.694 |
| Gradient Boosting | 0.8281 | Rs.1,159 |

## What's Inside
- 12 EDA plots covering all dataset features
- 18 engineered features from 10 raw columns
- 7 models compared with GridSearchCV tuning
- Price prediction tool with confidence intervals
- 8 documented challenges with solutions

## Tech Stack
Python, XGBoost, scikit-learn, pandas, matplotlib, seaborn

## Dataset
Provided as Flight_Fare.xlsx — domestic Indian flights 2019
